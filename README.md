# AI Threat Assessment Dashboard

Dashboard URL:

- [https://nickdulvy.github.io/AI-threat-assessment/threat_scoring_dashboard.html](https://nickdulvy.github.io/AI-threat-assessment/threat_scoring_dashboard.html)

Preferred browser:

- **Google Chrome** (recommended) or **Microsoft Edge**

## Quick Start (New User)

1. Open the dashboard URL above.
2. Click **Choose File** and select a local spreadsheet (`.csv`, `.xlsx`, or `.xls`).
3. Click **Load Rows** (or confirm the auto-loaded rows).
4. Confirm column mapping, then click **Start Threat/Reason Walkthrough**.
5. Score each row.
6. Use:
   - **Save Partial (Download)** while still working.
   - **Save Complete (Download)** when finished.

Saved filenames are datestamped:

- `<base>_PARTIAL_yymmdd_hhmm.csv`
- `<base>_COMPLETE_yymmdd_hhmm.csv`

## Autosave

- The dashboard autosaves your progress in your browser while you work.
- Reopening the same file structure restores your progress.
- You can continue later by loading your downloaded partial CSV.
